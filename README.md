## 📚 Conteúdo

### Pasta `teste-assistent-doce/`

Esta pasta contém exemplos e exercícios práticos de programação Python, organizados por temas:

#### 1. Debugging (`debug.py` e `explicacao_debug.md`)
- **Arquivo:** `debug.py` - Implementação de um calculador de compras com bugs intencionais.
- **Explicação:** `explicacao_debug.md` - Análise detalhada dos erros encontrados e suas correções.
- **Conceitos:** Entrada de dados, cálculos matemáticos, tratamento de erros, formatação de saída.

#### 2. Números Primos (`num_primos.py` e `explicacao_num-primo.md`)
- **Arquivo:** `num_primos.py` - Módulo completo para verificação e manipulação de números primos.
- **Explicação:** `explicacao_num-primo.md` - Documentação técnica das funções implementadas.
- **Funções:**
  - `eh_primo()`: Verifica se um número é primo
  - `listar_primos()`: Lista primos em um intervalo
  - `contar_primos()`: Conta primos em um intervalo
- **Conceitos:** Algoritmos eficientes, type hints, validação de entrada, testes.

#### 3. Refatoração (`refatoracao.py` e `explicacao_refatoracao.md`)
- **Arquivo:** `refatoracao.py` - Código refatorado para cálculo de estatísticas.
- **Explicação:** `explicacao_refatoracao.md` - Comparação entre código original e versão melhorada.
- **Conceitos:** Clean code, funções bem nomeadas, type hints, tratamento de erros.

## 🚀 Como Executar

### Pré-requisitos
- Python 3.6 ou superior
- Ambiente virtual (recomendado)

### Execução dos Exemplos

1. **Clone ou baixe o repositório**
2. **Navegue até a pasta do projeto**
3. **Execute os arquivos Python:**

```bash
# Para testar números primos
python teste-assistent-doce/num_primos.py

# Para testar estatísticas
python teste-assistent-doce/refatoracao.py

# Para o exemplo de debug (após correções)
python teste-assistent-doce/debug.py
```
